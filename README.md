# swift-chessgame


## specification



### 1. Board

- 8(rank)x8(file) 크기
- 가로줄 = rank
- 세로줄 = file



### 2. Pieces

- Pawn

- - 점수 - 1 점
  - 표기 - ♟(흑), ♙(백)

- Bishop

- - 점수 - 3 점
  - 표기 - ♝(흑), ♗(백)

- Knight

- - 점수 - 3 점
  - 표기 - ♞(흑), ♘(백)

- Luke

- - 점수 - 5 점
  - 표기 - ♜(흑), ♖(백)

- Queen

- - 점수 - 9 점
  - 표기 - ♛(흑), ♕(백)

- 빈곳

- - 표기 - .(점) 으로 표시
